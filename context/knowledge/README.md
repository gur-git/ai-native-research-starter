# context/knowledge/

The research substance: concept notes, paper notes, terrain maps — and, once the project reaches writing, the manuscript and its paragraph tree (the argument graph, provenance-stamped per node — `write` skill, G26). Agent-written, pitched to the researcher's profile; subfolders are fine where they help (e.g. `papers/` for deep-reads). Every file opens with a status line — `agent-drafted` until a gate passes, then `gate-stamped YYYY-MM-DD`. An unstamped note is just AI output; nothing builds on it until it carries a stamp.
