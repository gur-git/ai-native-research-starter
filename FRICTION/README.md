# FRICTION/

What about this way of working helped, chafed, was ignored, or felt like ceremony — the feedback channel back to the methodology, which is itself under study. The agent drafts dated entries here in the background (marked *agent-drafted*); the researcher adds notes of their own whenever something seems worth saying — no format required.

Conventions: one dated file per entry or batch (`YYYY-MM-DD-<slug>.md`); **about the methodology, never the research content**; each entry readable by a stranger with no context. The methodology's maintainers read this folder by arrangement during the early-user phase.
